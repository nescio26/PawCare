# PawCare 🐾

PawCare is a veterinary clinic management system designed to streamline pet care services, appointment scheduling, and clinic operations.

## 🚀 Features

- Pet management
- Owner management
- Appointment scheduling
- Admin dashboard

## 🛠️ Tech Stack

- Frontend: React (Vite), Tailwind
- Backend: Node.js / Express
- Database: MongoDB /

## 📦 Installation

```bash
npm install
npm run dev
```
